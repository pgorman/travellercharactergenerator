# Classic Traveller Character Generator #

This JavaScript character generator for the classic Traveller sci-fi roleplaying game follows the character creation rules in The Traveller Book (available in PDF on rpgnow.com/drivethrurpg.com).

Yes, characters _can_ die during character creation!

[Play with it now.](http://devilghost.com/software/travellercharacter/)

(Reload the page to generate another character.)
